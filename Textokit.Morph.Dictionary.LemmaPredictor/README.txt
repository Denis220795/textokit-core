Данный модуль предназначен для лемматизации не словарных слов,
основываясь на статистической модели преобразований слов словаря OpenCorpora.
Статистическая модель лежит в resouces/models. Используй IOModelUtil чтобы десериализовать или сериализовать
модель.
Все манипуляции с моделью выполняются при помощи класса ModelWordsExtractor.
